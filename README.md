# 🚀 AutoViral Engine

### AI-Powered Social Media Video Production System

AutoViral Engine is an end-to-end AI orchestration platform that transforms a single idea into a fully rendered, ready-to-publish social media video through autonomous workflow automation.

Built using n8n and multiple AI services, the system coordinates content generation, voice synthesis, visual creation, video assembly, and rendering into a seamless production pipeline.

---

## 🌟 Project Overview

Traditional content creation requires multiple tools, manual editing, repetitive tasks, and significant production time.

AutoViral Engine automates the entire process.

Starting with a simple idea, the workflow generates captions, scripts, voiceovers, AI visuals, animated video clips, and a final rendered video ready for publishing.

The workflow consists of 27+ orchestrated automation steps and demonstrates practical AI orchestration, workflow engineering, parallel processing, and scalable automation design.

## 🚀 How It Works

1. User provides a video idea.
2. AI generates engaging captions.
3. AI creates a narration script.
4. ElevenLabs generates voiceovers.
5. Flux generates visual assets.
6. Kling converts images into video clips.
7. Creatomate assembles captions, visuals, and audio.
8. Final video is rendered automatically.
9. Output becomes ready for publishing.
